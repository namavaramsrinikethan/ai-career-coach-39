# AI Resume Analyzer SaaS

A full-stack AI-powered resume optimization platform that analyzes resumes against real internship and job URL's, calculates ATS compatibility,
identifies missing skills, and generates a fully AI-enhanced resume with downloadable PDF reports.

Upload your Resume PDF + paste a job URL → receive ATS scores, skill-gap analysis, project recommendations, and a side-by-side comparison between your 
original and AI-optimized resume within seconds.

## 🖥️ Frontend Experience

The platform provides a modern SaaS dashboard designed specifically for students and job seekers.

### User Workflow

1. Sign Up / Login
2. Upload Resume PDF
3. Paste Unstop Job/Internship URL
4. Run AI Analysis
5. Review Results Dashboard
6. Compare Original Resume vs AI-Modified Resume
7. Download Reports and Optimized Resume

### Dashboard Features

* Clean and responsive user interface
* Secure authentication system
* Resume upload functionality
* Real-time analysis progress tracking
* ATS score visualization
* Skills match analytics
* Missing skills breakdown
* AI-generated recommendations
* Side-by-side resume comparison
* PDF download options

### Resume Comparison Interface

One of the platform's core frontend features is the interactive comparison view.

Users can visually compare:

* Original Resume
* AI-Modified Resume

The comparison highlights:

* Added ATS keywords
* Missing skills addressed by AI
* Improved project descriptions
* Enhanced certifications and skills
* Resume optimization suggestions

This allows users to clearly understand every modification before downloading the final optimized resume.
